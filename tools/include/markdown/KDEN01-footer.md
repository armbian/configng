=== "Display Manager"

    KDE Neon uses **SDDM** as its default display manager.

=== "Session files"

    - `/usr/share/wayland-sessions/plasma.desktop`
    - `/usr/share/xsessions/plasmax11.desktop`

=== "Autologin"

    Autologin configuration is stored in:

    - `/etc/sddm.conf.d/autologin.conf`
