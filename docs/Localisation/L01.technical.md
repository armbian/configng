# L01

**description:** Change Locales reconfigure the language and character set

**Command:** 
~~~
dpkg-reconfigure locales, source /etc/default/locale ; sed -i "s/^LANGUAGE=.*/LANGUAGE=$LANG/" /etc/default/locale, export LANGUAGE=$LANG
~~~
