# Distribution upgrades


***

## Upgrade to latest stable / LTS
Release upgrade is irriversible operation which upgrades all packages. 

Resoulted upgrade might break your build beyond repair!

**Command:** 
~~~
armbian-config --cmd SY101
~~~

**Author:** @igorpecovnik

**Status:** Stable
